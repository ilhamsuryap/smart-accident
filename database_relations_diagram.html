⚠️ Important Notes:
  • Auto-Assignment: Kecelakaan Preprocessing secara otomatis diassign ke Segmen Jalan terdekat berdasarkan perhitungan perpendicular distance (tolerance ~50m)
  • Rekap Segmen: Unique constraint pada (segmen_jalan, periode_tahun) untuk mencegah duplikasi rekapitulasi per tahun
  • Z-Score Analysis: Dihitung per Ruas Jalan dengan interval dinamis berdasarkan standar deviasi
  • Geometry Storage: Disimpan dalam format GeoJSON (LineString) untuk visualisasi peta
  • Koordinat Precision: Menggunakan 20 decimal places (1.1cm precision) untuk akurasi maksimal
🔒 Data Integrity Rules:
  • Cascading Deletes: Menghapus Ruas Jalan akan menghapus semua Segmen Jalan terkait
  • SET_NULL on Delete: Menghapus Polres akan set polres_id menjadi NULL (data tetap ada)
  • Unique Constraints: Email User dan Kode Polres harus unik
  • Validators: panjang_km dan kerugian_materi minimal 0 (non-negative)
  • Signal Handlers: Membuat Profile otomatis ketika User dibuat (OneToOne relationship)

📋 Detailed Table Specifications

1. User (coreapp_user)

Authentication & Authorization - Primary user model with custom manager for role-based access

2. Profile (coreapp_profile)

User Profile - Extended profile with institution, position, and Polres assignment

3. Polres (coreapp_polres)

Police Headquarters - Dynamic Polres management by superadmin with unique code

4. Ruas Jalan (coreapp_ruasjalan)

Road - Road segments with geometry (GeoJSON LineString) and classification

5. Segmen Jalan (coreapp_segmenjalan)

Road Segment - Subdivisions of roads with kilometer markers and coordinates

6. Kecelakaan Preprocessing (coreapp_kecelakaanpreprosesing)

Accident Data - Preprocessed accident records auto-assigned to road segments

7. Rekap Segmen (coreapp_rekapsegmen)

Segment Summary - Aggregated accident statistics per segment per year

8. Analisis Z-Score (coreapp_analisiszscore)

Risk Analysis - Z-Score statistical analysis for accident risk categorization

9. Social Account (socialaccount_socialaccount)

OAuth Integration - Django-allauth social authentication provider mapping
