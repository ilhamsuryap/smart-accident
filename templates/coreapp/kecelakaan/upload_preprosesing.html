Format File

File harus berisi kolom-kolom berikut:

  • latitude (format: decimal)
  • longitude (format: decimal)
  • korban_meninggal (angka)
  • korban_luka_berat (angka)
  • korban_luka_ringan (angka)
  • kerugian_materi (angka)
  • desa (text)
  • kecamatan (text)
  • kabupaten_kota (text)
  • keterangan (text)
