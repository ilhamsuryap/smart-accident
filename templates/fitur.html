Fitur Analisis pada Smart Accident

Smart Accident menyediakan fitur pengelolaan data kecelakaan, analisis lokasi rawan, pengelompokan faktor penyebab, serta analisis pola waktu kecelakaan untuk membantu penyajian informasi yang lebih terstruktur.

Fitur Utama Sistem

Setiap fitur dirancang untuk membantu admin mengolah data kecelakaan menjadi informasi yang mudah dianalisis dan digunakan sebagai dasar rekomendasi penanganan.

database

Pengelolaan Data Kecelakaan

Fitur ini digunakan untuk menyimpan, mengubah, menghapus, dan mengelola data kecelakaan agar data tersusun rapi sebelum dilakukan proses analisis.

monitoring

Analisis Z-Score

Digunakan untuk membantu mengetahui tingkat kerawanan kecelakaan berdasarkan nilai standar sehingga lokasi dengan risiko tinggi dapat diidentifikasi.

account_tree

Analisis Faktor Penyebab dengan AHC

Agglomerative Hierarchical Clustering digunakan untuk mengelompokkan data kecelakaan berdasarkan karakteristik yang serupa sehingga faktor penyebab dapat dianalisis lebih jelas.

schedule

Analisis Pola Waktu dengan K-Means

K-Means Clustering digunakan untuk mengelompokkan data kecelakaan berdasarkan pola waktu kejadian sehingga periode rawan kecelakaan dapat diketahui.

bar_chart

Visualisasi Hasil Analisis

Hasil analisis disajikan dalam bentuk tabel, grafik, dan visualisasi agar informasi lebih mudah dibaca dan dipahami oleh admin.

lightbulb

Rekomendasi Penanganan

Sistem membantu menyajikan rekomendasi penanganan berdasarkan hasil analisis untuk mendukung upaya pencegahan kecelakaan lalu lintas.

Fitur Analisis Dikelola oleh Admin

Untuk menjaga keamanan dan validitas data, fitur pengelolaan data dan analisis hanya dapat digunakan oleh admin yang memiliki hak akses pada sistem.
